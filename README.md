# NEXUS OS — Unified Business Management Platform

🚀 **Одна платформа для любого бизнеса**

## 📊 Вертикали

### 🛍️ Retail (Магазины)
- POS-касса со штрих-кодами
- Управление товарами и остатками
- Аналитика продаж
- Клиентская база

### ☕ HoReCa (Рестораны & Кафе)
- POS-касса
- Управление меню
- Программа лояльности
- Kitchen Display System

### 🏭 Manufacturing (Производство)
- Управление производством
- Отслеживание оборудования
- Аналитика
- Digital Twin (AI предсказания)

## 🛠️ Tech Stack

- **Backend:** Google Apps Script
- **Database:** Google Sheets
- **Frontend:** Vue.js
- **Hosting:** Google Cloud Functions / Netlify
- **Analytics:** Google Looker Studio

✅ Универсальная система управления
